Integrate graphtor-docs as a new capability pack in autoharness, following the established pattern used by agent-intercom, agent-engram, and backlogit. graphtor-docs is a local-first documentation RAG system that indexes docs into an embedded graph+vector store and serves them to AI agents via 8 MCP tools (search_local_docs, search_semantic, research_topic, traverse_doc_links, list_sources, get_chunk_by_id, get_document, get_status). The pack deepens Primitive 1 (knowledge retrieval via indexed docs) and Primitive 9 (repository knowledge augmented with external doc sources). Integration touches: schemas (enum registration + config block + profile detection), templates (instruction template + agent overlays), skills (install-harness, tune-harness, workspace-discovery), verification (verify_workspace.py + tests), and documentation (harness-architecture).
