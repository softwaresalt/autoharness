title: Add community_templates tracking to harness-manifest schema
updated_at: 2026-05-09T08:52:04.9872642-07:00
---

Add a community_templates property to schemas/harness-manifest.schema.json. Structure: community_templates is an optional array of objects, each with: template_id (string, matches registry entry), template_path (string, path to .tmpl in autoharness_home), installed_path (string, path where it was installed in target workspace), installed_at (string, date-time), checksum (string, SHA-256 of installed artifact). This enables the tuner to detect whether installed community templates have drifted from their source. Also update the versioned schemas in schemas/harness-manifest/ if applicable.
