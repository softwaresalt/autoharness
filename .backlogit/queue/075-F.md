Deterministic, reusable overlay that makes agents ROUTE retrieval through installed packs (engram for code/symbol/semantic; graphtor-docs for docs/domain/business-context) instead of silently defaulting to grep/web. Two surfaces: a SOFT session-time Pre-Retrieval Routing coordinator instruction (agents self-apply it) and a HARD install/CI-time verifier that deterministically checks the routing discipline is INSTALLED and COHERENT. autoharness cannot intercept live tool choices, so actual per-call usage is NOT a testable acceptance condition; the deterministic guarantee is that a workspace cannot silently drop the discipline without the verifier failing.
