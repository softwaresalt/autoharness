Add a review comment resolution loop to ship.agent.md.tmpl after the PR is created in Step 4 (PR Lifecycle). The loop: (1) Poll for Copilot review comments using gh api graphql reviewThreads query, filtering by isResolved=false. (2) For each unresolved thread, read the comment, apply the fix, commit, push. (3) Reply to the comment via addPullRequestReviewComment mutation. (4) Resolve the thread via resolveReviewThread mutation. (5) Re-poll after all fixes to check for new comments triggered by the fix commit. (6) Repeat until zero unresolved threads remain OR circuit breaker (3 cycles) is hit. (7) Only then present the PR as ready for operator merge approval. Include the GraphQL queries as inline code blocks. Document bot detection: match author.login containing 'copilot-pull-request-reviewer' for Copilot comments; treat other bot comments and human comments separately.
